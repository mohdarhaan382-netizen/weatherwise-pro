# 🌦️ WeatherWise Pro

## Smart Weather Forecast Application

WeatherWise Pro is an advanced real-time weather forecasting web application built using frontend technologies. It provides users with current weather information, forecasts, and weather-related insights through a modern and responsive interface.

---

## 🚀 Features

- 🌍 Search weather by city
- 📍 Detect current location weather
- 🌡️ Real-time temperature updates
- 💧 Humidity information
- 🌬️ Wind speed details
- 🌡️ Atmospheric pressure
- 📅 7-day weather forecast
- 📊 Temperature trend visualization
- 🌙 Dark mode support
- 📱 Fully responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript ES6
- OpenWeather API
- Chart.js
- Font Awesome
